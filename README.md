# Teste Automatizado - Site do Tesouro Direto. #
Teste realizado no site do Tesouro Direto foi feito para o projeto que será apresentado no S2B PUCRS 2018/1.
Foram utilizados os seguintes conceitos: 

* AppObjects
* Framework
* Tasks
* TestCases
* TestSuites
* Reports

### Ferramentas:
* Eclipse IDE - eclipse.org
* Maven - maven.apache.org
* JDK 8 - oracle.com

### Bibliotecas externas
Dependências obtidas através do gerenciador de dependências do Maven.

* Selenium WebDriver - seleniumhq.org
* WebDriver Manager - github.com/bonigarcia/webdrivermanager
* JUnit - junit.org
* Extent Reports - extentreports.com
